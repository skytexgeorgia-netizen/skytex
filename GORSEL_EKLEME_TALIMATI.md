# Görsel Ekleme Talimatı

## ⚠️ ÖNEMLİ: Görseller Henüz Eklenmedi

Web sitesi görselleri göstermek için hazır, ancak görseller henüz `public` klasörüne eklenmedi. 

## Görselleri Nasıl Ekleyeceksiniz?

### 1. Logo Görseli
**Dosya Yolu:** `public/logo/skytex-logo.png`

- SKYTEX logosunu PNG formatında kaydedin
- Şeffaf arka plan önerilir
- Boyut: 48x48px - 96x96px (retina için 2x)

### 2. Ana Sayfa Hero Banner
**Dosya Yolu:** `public/factory/hero-banner.jpg`

- Fabrika veya üretim görseli
- Boyut: 1920x1080px veya daha büyük
- Format: JPG

### 3. Hakkımızda Sayfası Görselleri
**Dosya Yolları:**
- `public/factory/factory-1.jpg`
- `public/factory/factory-2.jpg`

- Fabrika içi veya makine görselleri
- Boyut: 1200x800px
- Format: JPG

### 4. Portfolyo - Üretim Tesisleri
**Dosya Yolları:**
- `public/factory/machine-1.jpg`
- `public/factory/machine-2.jpg`
- `public/factory/machine-3.jpg`
- `public/factory/machine-4.jpg`

- Barudan makineleri veya üretim görselleri
- Boyut: 1200x800px
- Format: JPG

## Hızlı Test İçin

Görselleri ekledikten sonra:
```bash
npm run dev
```

Tarayıcıda `http://localhost:3000` adresini açın ve görsellerin göründüğünü kontrol edin.

## Şu Anki Durum

- ✅ Kodlar hazır ve görselleri gösterecek şekilde yapılandırıldı
- ✅ Görseller yoksa placeholder'lar gösteriliyor
- ⏳ Görselleri `public` klasörüne eklemeniz gerekiyor

## Görselleri Ekledikten Sonra

Görselleri ekledikten sonra:
1. Web sitesini yenileyin (F5)
2. Görsellerin doğru göründüğünü kontrol edin
3. Responsive tasarımı test edin (mobil/tablet/desktop)
